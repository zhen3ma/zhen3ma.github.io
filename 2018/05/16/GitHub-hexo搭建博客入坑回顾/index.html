爱好写作，崇尚技术，喜欢创新，爱瞎鼓捣。

为什么是github+hexo?

免费 开源 热爱 喜欢

发现github

github开启了自己对于开源，前端，科学上网，博客这些兴趣的探索。
在自己看来，github就是一个技术宅的朋友圈。
作为菜鸟，我是只看别人朋友圈的那个。
如果你有兴趣想深入了解一下它
[github可以做什么？]https://www.zhihu.com/topic/19566035/hot

hexo背景

hexo出自台湾大学生 tommy351 之手，是一个基于Node.js的静态博客程序，其编译上百篇文字只需要几秒。hexo生成的静态网页可以直接放到GitHub Pages，BAE，SAE等平台上。

不过，幸运的是，如果我们抛开技术，原理层面上，单纯的用github搭建一个博客，貌似会变得简单起来。
